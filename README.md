
# Aguas Las Mercedes - Prototipo React

Proyecto en Vite + React listo para deploy en Vercel.

## Cómo desplegar (local)
1. `npm install`
2. `npm run dev` (desarrollo)
3. `npm run build` (producción)
4. `npm run preview` (ver build localmente)

## Notas
- La app usa LocalStorage para guardar pedidos (sin backend).
- Botón 'Ver Ruta' abre Google Maps con la dirección.
- Para integraciones reales (pagos, base de datos) hay que añadir backend o servicios externos.
